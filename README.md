# Project Alpha Cybersecurity Repository

This repository hosts a collection of advanced cybersecurity projects and research.

## Active Projects

* **[AI-CASB](AI_CASB/)**: Agentic AI Security Gateway - A local, self-hosted security layer that intercepts AI coding agent traffic (like GitHub Copilot, Continue, Cline) to protect against prompt injections, data exfiltration, and other LLM-specific threats securely at the perimeter proxy level.

Please explore the individual project sub-directories for specific documentation, implementation details, and deployment instructions.
